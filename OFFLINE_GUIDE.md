# 🌳 Deforestation Prediction Project - Offline Guide

This project is now fully self-contained and can be run without internet access (except for the initial package installation).

## 🚀 How to Run
Simply double-click the **`RUN_DASHBOARD.bat`** file. 

This script will:
1. Automatically set up your Python environment.
2. Install any missing libraries.
3. Launch the interactive dashboard in your browser.

## 📁 Project Structure
- `gui.py`: The main application code.
- `data/data.csv`: Historical dataset used for analytics and training.
- `requirements.txt`: List of necessary Python libraries.
- `RUN_DASHBOARD.bat`: One-click launcher for Windows.

## 🛠️ Requirements
- Python 3.9 or higher installed on your system.
- All machine learning models are trained locally on your data at startup, so no external APIs are needed.

## 🤖 Features
- **Analytics Dashboard**: View trends and risk distributions.
- **Predictor**: Predict risk levels using a local Random Forest model.
- **Image Analysis**: Scan satellite images for deforestation patterns offline.
- **AI Assistant**: A rule-based expert guide for environmental solutions.
